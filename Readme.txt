Pada Episode 4 ini mempelajari tentang salah satu fitur dari React Hook yaitu useState, yang dimana
dengan menggunakan useState kita dapat mengembalikan value state terbaru dan sebuah fungsi untuk mengubah
value state tersebut. Karena useState memiliki syntax yang menerima satu paramater, yaitu nilai awal state

contoh :
    const [age, setAge] = useState(23);
